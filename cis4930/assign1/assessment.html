What does HTML stand for?

What is the purpose of CSS?

Incorrect. Structuring HTML is the job of HTML itself, not CSS. Read About It
Correct! CSS is used to style and layout HTML elements.
Incorrect. Writing scripts for web pages is the role of JavaScript. Read About It

What is semantic HTML?

Incorrect. Semantic HTML goes beyond comments for developers. Read About It
Correct! Semantic HTML uses meaningful tags to improve accessibility and readability.
Incorrect. Semantic HTML is human-readable as well. Read About It

Which tag is used to define metadata?

Correct! The <meta> tag is used to define metadata in an HTML document.
Incorrect. The <title> tag defines the title of the document, not metadata. Read About It
Incorrect. The <header> tag defines a section header, not metadata. Read About It

What is the purpose of the <body> tag?

Incorrect. The <body> tag does not define metadata; that's the <meta> tag's job. Read About It
Correct! The <body> tag contains the visible content of the page.
Incorrect. The <body> tag does not structure the head section; that’s the role of the <head> tag. Read About It
