This assignment introduced the foundational concepts of HTML and CSS. The process of manually writing HTML and CSS provided valuable insight into how these technologies interact and taught me the importance of proper file management for web development projects.

  • Understand the structure of an HTML document.
  • Learn to use CSS for styling and layout.
  • Practice linking files and creating relative paths.
  • Develop organizational skills for web development projects.
