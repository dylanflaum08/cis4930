Assignment 1

This assignment introduced the foundational concepts of HTML and CSS. I learned how to create web pages from scratch, structure them using tags, and style them with CSS to achieve a professional appearance. Additionally, I practiced organizing files and using relative paths to link resources effectively.

The process of manually writing HTML and CSS provided valuable insight into how these technologies interact and taught me the importance of proper file management for web development projects.

  • Understand the structure of an HTML document.
  • Learn to use CSS for styling and layout.
  • Practice linking files and creating relative paths.
  • Develop organizational skills for web development projects.

Assignment 2

This assignment focused on using decision control structures in JavaScript and working with web forms. I learned how to collect user input using various form elements, validate the data using regular expressions, and dynamically process user input to generate personalized feedback. This enhanced my understanding of the Document Object Model (DOM) and how JavaScript interacts with HTML elements.

The project also introduced responsive design challenges, where I had to ensure that the form and outputs adapted well across different screen sizes. Implementing if-else conditions and using event-driven programming provided deeper insights into interactive web development.

  • Learn how to create and validate web forms.
  • Understand decision control structures in JavaScript.
  • Work with the Document Object Model (DOM) to manipulate HTML dynamically.
  • Apply CSS techniques to enhance form usability and responsiveness.

Assignment 3

This assignment focused on repetition control structures and arrays as a data structure. I implemented a program that collects a list of numbers from a user and computes several different descriptive statistics, such as mean, median, mode, and standard deviation.

I also created a separate JavaScript file, `statistics.js`, which contains a function library for performing statistical calculations. The assignment emphasized modularity and code reusability by structuring the functions efficiently.

  • Implemented loops and array manipulations in JavaScript.
  • Created reusable functions for statistical calculations.
  • Structured a web form to collect and process numerical data.
  • Displayed statistical results dynamically using JavaScript.

Additionally, unit testing was conducted with three different datasets to verify the accuracy of the statistical calculations. The results of the testing have been documented in an HTML table for reference.

Assignment 4

This assignment focused on integrating **Bootstrap and jQuery** to enhance the **responsiveness and interactivity** of a web form. The original web form from Assignment 2 was **completely redesigned** using Bootstrap's grid system, form controls, and buttons to create a clean and responsive layout.

In addition, I used **jQuery** to handle form validation, event handling, and dynamic updates to the page. Instead of manually manipulating the DOM using `document.getElementById`, I leveraged jQuery's powerful `$()` selector and event-driven programming model. I also added smooth animations using `.fadeIn()`.

  • Implemented Bootstrap for a responsive layout with grid and form controls.
  • Used jQuery for form validation, event handling, and dynamic content updates.
  • Created a smoother user experience with animations and error messages.
  • Ensured compatibility across different screen sizes and devices.

Additionally, I completed a **Bonus Challenge** where I recreated the form using **W3.CSS** as an alternative to Bootstrap and **AngularJS** as an alternative to jQuery. This gave me insight into how different frameworks handle styling and interactivity.
